# Fit-Connect

Fit Connect is an interactive platform that allows users to share their fitness journey with others through photos, videos, stories, and more. Whether it's a workout video, progress updates, or motivational stories, Fit Connect keeps you engaged and motivated.

## Features

- Share photos, videos, and stories
- Like, comment, and share posts
- Real-time chat with other users
- Personalized feed and notifications
- User profile and activity tracking

## Tools & Technologies Used

- **Frontend:** React, TypeScript, JavaScript
- **Backend:** Node.js, Express (backend work in progress, updates coming soon)
- **Database:** SQL, MongoDB
- **Others:** DNB (Database Notation Builder)

## Setup Instructions

1. Clone the repository.
2. Install the dependencies: `npm install`
3. Start the development server: `npm start`

This version highlights that the backend work is ongoing and will be updated soon!
